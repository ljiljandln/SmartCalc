Project SmartCalc
To install the calculator use the ”install” goal. The executable
/SmartCalc file is created in the ”release” directory.
To archieve the executable file use the ”dist” goal.
To delete the executable file and the archieve use the ”uninstall”
goal. To delete the rest of the temporary files use the ”clean” goal.
The ”test” goal is used to start the unit tests. The ”gcovreport”
goal creates a coverage report.
To use the calculator enter the equasion and press ”=”. To calculate
an expression with an ”x” enter the x in the special field with the ”x”
box checked. To draw a graphic enter an equasion with an x, enter
the range in the x field and click draw.
